## Description

CRUD and Auth API for Recruiting

## Installation

```bash
$ npm install
```

## Running the app

```bash

$ docker container start nest-users

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
